#   Job Application Tracker

Track, manage, and stay on top of your job applications - all in one place. 

Description
--------------
 • A full-stack web application that allows users to organize, track, and manage job applications in one place. Users can add new applications, update their status, and keep their job search information structured and easily accessible.

Purpose
---------------
• I built this application to solve a real problem I experienced while applying to jobs, keeping track of multiple applications across different platforms. This tracker provides a centralized place to manage applications, statuses, and important details without relying on scattered emails or job boards.

Features
---------------
• Add and store job applications
• Update application status 
• Edit existing applications
• Delete applications
• View all applications in a structured table
• Track follow-up dates and notes

Tech Stack
-----------------
backend
• Java
• Spring Boot
• REST API

Database
• PostgreSQL

Frontend
• HTML
• CSS
• JavaScript

How It Works
-----------------
• The Frontend sends HTTP requests (GET, POST, PUT, DELETE) to the backend API
• The backend processes requestts and interacts with the PostgreSQL database
• Data is returned as JSON and dynamically rendered in the UI

Future Improvements
------------------------
• Filter and search functionality
• Authentication (user accounts)
• Dashboard with analytics
• Send automated email reminders for follow-up dates
• Responsive design improvements
